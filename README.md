### 거래글 상세

- `참여하기` 를 이용해 거래 참여 가능
  - 거래에 참여하는 유저 명 수, 인원별 가격, 거래 장소 안내
  - 거래에 참여한 다른유저들에게 `앱 푸시 알림`
